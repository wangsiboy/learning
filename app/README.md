# 综合运用

1. 定义app
npm init 
2. express框架
npm install -g express-generator
3. 模版引擎
npm install handlebars

## 后端

* nodejs驱动
* express框架
* [handlebars](http://handlebarsjs.com/) 模版引擎
* mongoDB 数据库
* [mongoose](http://mongoosejs.com/) 数据库建模

## 前端

* jquery
* sass

## NPM 安装的模块

* Grunt 自动化工具
* Moment.js 时间日期格式化模块
