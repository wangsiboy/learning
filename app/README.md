# 综合运用

整一个项目的架子，所有步骤纪录在这。

## 后端

* nodejs驱动

npm init 

* express框架

npm install -g express-generator

* mongoDB 数据库

* [mongoose](http://mongoosejs.com/) 数据库建模

## 前端

* bower管理前端包

npm install bower -g

加一个.bowerrc文件
{
  "directory": "public/vendor"
}
bower init
bower install --save bootstrap-sass

https://github.com/twbs/bootstrap-sass#c-bower

* connect-bundle
